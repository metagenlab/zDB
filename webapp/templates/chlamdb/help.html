Search for a protein of interest

The following protein accessions can be used to search the database:
Name Example

RefSeq locus tag WCW_RS07680

UniParc accession UPI0001D5C1DD

UniProtKB-ID D6YS95_WADCW

Accessions from KEGG, COG, Pfam and InterPro can also be searched to retreive the list of proteins with the corresponding annotation:
Accession type Example Description

KEGG modules M00023 Amino acid metabolism
